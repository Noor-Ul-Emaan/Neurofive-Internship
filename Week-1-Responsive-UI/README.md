# Week 1: Responsive UI Landing Page

## Project Choice
- Built a fully responsive landing page/homepage for a product.

## Framework & Structure Choice
- **React + Vite**: Fast performance, quick build setup, and robust state management.
- **Component-based architecture**: Reusable blocks such as `Navbar`, `Hero`, and `Features` components to handle sections cleanly.
- **Responsiveness**: Handled breakpoints smoothly across mobile, tablet, and desktop views.
