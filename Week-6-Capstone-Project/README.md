# Portfolio-Defining Project: Task & Project Management System

## Case Study
- **Problem Statement:** Teams require an intuitive platform to create, manage projects, and assign tasks with secure access controls.
- **Tech Stack Choices:** MongoDB, Express.js, React (Vite), Node.js (MERN), JWT Authentication, Helmet security, RESTful APIs.
- **Challenges & Solutions:** Resolving JWT expiration and state persistence handled via protected route wrappers and secure localStorage management.
